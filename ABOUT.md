# GRIT.ON 앱 소개

> 클래식 음악가를 위한 AI 연습 코치

---

## 앱 개요

**GRIT.ON**은 클래식 음악을 연습하는 사람들을 위한 스마트 연습 도우미 앱입니다. 마치 개인 코치가 옆에서 연습을 지켜보듯, 연습 시간을 자동으로 측정하고 분석해줍니다.

### 핵심 가치

- **정확한 연습 시간 측정**: 실제로 악기를 연주한 시간만 측정 (대화, 휴식 시간 제외)
- **AI 기반 분석**: 어떤 부분이 취약한지 자동으로 파악
- **체계적인 연습 관리**: 목표 설정부터 기록까지 한 곳에서

## 주요 화면 설명

### 1. 홈 화면

앱을 열면 가장 먼저 보이는 화면입니다.

| 요소 | 설명 |
|------|------|
| **인사말** | 시간대에 따라 "좋은 아침이에요" 등으로 변경 |
| **통계 카드** | 총 연습시간, 이번 주 세션 수, 연속 연습일 표시 |
| **오늘의 목표** | 하루 목표 대비 진행률을 원형 그래프로 표시 |
| **명언 카드** | 유명 음악가들의 명언으로 동기부여 |
| **연습 시작 버튼** | 누르면 바로 연습 화면으로 이동 |
| **집중 구간** | AI가 분석한 취약한 마디를 색깔로 표시 |

### 2. 연습 화면

실제로 연습을 진행하는 화면입니다.

| 요소 | 설명 |
|------|------|
| **곡 선택** | 연습할 곡을 선택 (탭하면 목록 표시) |
| **타이머** | 큰 숫자로 순연습시간 표시 |
| **파형 그래프** | 소리에 반응하여 움직이는 막대 그래프 |
| **입력 레벨** | 현재 소리 크기를 게이지로 표시 |
| **재생/일시정지/종료** | 연습 제어 버튼 |

**순연습시간이란?**
> 실제로 악기에서 소리가 나는 시간만 측정합니다. 악보를 보거나 쉬는 시간은 제외되어 진짜 연습량을 정확히 알 수 있습니다.

### 3. 녹음 기록 화면

지금까지의 연습 기록을 확인할 수 있습니다.

| 요소 | 설명 |
|------|------|
| **통계 요약** | 총 녹음 수, 평균 점수, 총 연습시간 |
| **녹음 목록** | 날짜별로 정리된 연습 기록 카드 |
| **상세 보기** | 카드를 탭하면 해당 연습의 상세 분석 확인 |

### 4. 녹음 상세 화면

개별 연습 세션의 자세한 분석 결과입니다.

| 요소 | 설명 |
|------|------|
| **연습 정보** | 곡명, 날짜, 연습 시간 |
| **점수** | 전체 점수, 정확도, 템포 안정성, 다이나믹 |
| **AI 분석** | 잘한 점과 개선할 점 요약 |
| **집중 구간** | 더 연습이 필요한 마디 목록 |

### 5. AI 분석 화면

전체적인 연습 현황과 AI의 조언을 볼 수 있습니다.

| 요소 | 설명 |
|------|------|
| **현재 곡 정보** | 연습 중인 곡과 작곡가 |
| **성장 추이** | 점수가 얼마나 향상되었는지 |
| **마디별 분석** | 32마디를 색깔로 구분 (주황=집중 필요, 파랑=양호) |
| **AI 추천** | 구체적인 연습 방법 제안 |
| **점수 그래프** | 최근 연습의 점수 변화 |

### 6. 연습 계획 화면

체계적인 연습을 위한 계획 관리 화면입니다.

| 요소 | 설명 |
|------|------|
| **주간 캘린더** | 요일별 연습량을 색 진하기로 표시 |
| **오늘의 계획** | 체크리스트 형태의 할 일 목록 |
| **진행률 바** | 오늘 계획 완료율 표시 |
| **AI 추천** | 오늘 집중하면 좋을 연습 제안 |

### 7. 프로필/설정 화면

개인 설정과 계정 관리를 할 수 있습니다.

| 요소 | 설명 |
|------|------|
| **프로필** | 이름, 악기, 레벨 정보 |
| **구독 상태** | Free 또는 Pro 멤버십 |
| **일일 목표** | 하루 연습 목표 시간 설정 (15분~120분) |
| **알림 설정** | 연습 알림 켜기/끄기 |
| **언어 설정** | 한국어, English, 日本語, 中文 |
| **연습 통계** | 총 연습 시간, 녹음 수 등 |

## 기술적 특징 (쉬운 설명)

### PWA (Progressive Web App)

- **의미**: 웹사이트지만 앱처럼 사용 가능
- **장점**: 앱스토어 없이 홈 화면에 추가 가능
- **특징**: 오프라인에서도 일부 기능 사용 가능

### 순연습시간 측정 원리

1. **마이크 접근**: 연습 시작 시 마이크 권한 요청
2. **소리 감지**: 일정 크기 이상의 소리만 인식
3. **노이즈 필터링**: 짧은 잡음(기침, 책 넘기는 소리)은 무시
4. **시간 누적**: 연속적인 소리가 있을 때만 시간 카운트

### 데이터 저장

- **로컬 저장**: 연습 기록이 기기에 자동 저장됨
- **오프라인 지원**: 인터넷 없이도 연습 기록 가능
- **동기화 대기**: 나중에 인터넷 연결 시 서버와 동기화 예정

## 앱 구조

```
GRIT.ON
├── 홈 (/)
│   └── 대시보드, 통계, 오늘의 목표
│
├── 연습 (/practice)
│   └── 타이머, 녹음, 소리 감지
│
├── 녹음 (/recordings)
│   ├── 녹음 목록
│   └── 녹음 상세 (/recordings/[id])
│
├── 분석 (/analysis)
│   └── AI 분석 리포트
│
├── 계획 (/plans)
│   └── 주간 계획, 오늘의 할 일
│
└── 프로필 (/profile)
    └── 설정, 계정 관리
```
